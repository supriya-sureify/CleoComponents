# CleoComponents

Component library designed with typescript, tailwind

## Getting started

Clone the repo

```
git clone https://github.com/supriya-sureify/CleoComponents.git
```

Install the dependencies

```
npm install
```

Start the dev server

```
npm run dev
```

Run the build

```
npm run build
```

Locally preview the production build

```
npm run preview
```
